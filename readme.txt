=================================================
　　ティラノライダーでホットキーリロードプラグイン
　　作った人：@violet_ncnc
=================================================

【はじめにお読みください】
このプラグインは、ティラノライダーでのゲーム開発時にホットキーによるリロードを行うためのものです。
ティラノスクリプトv471fおよびティラノライダーv220にて動作確認しています。


【使い方】
このテキストが入っているフォルダ（hotreload）ごとティラノスクリプトの「data/others/plugin」フォルダに入れてください。
その後、first.ksに以下のとおり記述してください。

==========================
記述例：
--------------------------
[plugin name="hotreload"]
==========================
上記のタグを記述した時点から、後述するホットキーを使用できるようになります。
また、指定できる引数は以下のとおりです。

==========================
key  ：ショートカットキーを指定する文字列
--------------------------
例：[plugin name="hotreload" key="Shift+Alt+R"]
各キーの指定方法については↓を参照するといいかも
参照URL：http://docs.nwjs.io/en/latest/References/Shortcut
==========================


【ホットキーによるリロード】
デフォルトでは「Shift+Alt+R」（3つのキーを同時押し）で、ティラノライダーの「ゲーム開始」ボタンをクリックしたのと同様の操作を行います。
通常のショートカットキー割当では、ティラノライダーのメイン画面にフォーカスがあたっていないと動作しませんが、ホットキーによるリロードでは、
たとえばスクリプトを編集するエディタから直接ゲーム画面のリロードをする、といった操作が可能となります。


【注意事項】
このプラグインは、ティラノライダーを使ったゲーム開発用のプラグインです。
製品版としてリリース時にはプラグインおよびプラグイン読み込み用のタグを削除してください。
